Welcome to the Gradient Viewer! This is a service that allows you to isolate and play around with the cool gradients on the new Genius pages.

In order to use this tool properly, you need to add a bookmarklet to your browser with the following text as the link (click to copy to clipboard): 

javascript:function objectToURLParams(e){let t=new URLSearchParams;for(let r in e)e.hasOwnProperty(r)&&t.append(r,e[r]);return t.toString()}if(confirm("Are you SURE you want to use the Gradient Viewer on this song? (This will open in a new tab.)")){let e=objectToURLParams({name:document.querySelector('h1[class*="SongHeader-desktop"]').textContent,artist:_sf_async_config.authors,gradient:window.getComputedStyle(document.querySelector("div[class*=PageGrid-desktop-sc-]"))["background-image"],img:btoa(document.querySelector("img[class*=SizedImage-sc]").src),href:window.location.pathname});window.open("%WEBSITE%?"+e,"")}

Once you've done that, simply go to the page in question and click the bookmark! It will take you back here, but the page will visibly look a lot more different than it does now...
